Meet in the Middle

“友達とどこで会う？” を一瞬で解決。
出発駅を入れるだけで最適な集合地点を提案します。

01. 比重スライダー

“今日は相手寄りに” など
スライダーで簡単に重み付け。

02. キーワード検索

「カフェ」「カラオケ」など
目的に合わせた駅と POI を表示。

03. URL シェア

LINE で URL を共有すれば
参加者が増えてもリアルタイム更新。

04. 交通時間考慮

Google Maps API と時刻表 API で
所要時間ベースの提案を実装予定。
